## 📊 進捗計算方法

### 計算式

```
Phase進捗(%) = (完了タスクの重み合計 / 全タスクの重み合計) × 100
```

### 例（Week 1完了時）

```
Phase 1a:
- 完了: UI-001(10%) + UI-002(10%) + UI-003(5%) + UI-004(5%) = 30%
- Phase 1全体: 30%

全体進捗:
- Phase 1: 30%
- 全体: 30% × 25% = 7.5%
```

## 📈 進捗更新ルール

### 更新タイミング

1. **タスク完了時**: 即座に更新
2. **マイルストーン達成時**: Phase全体を再計算
3. **週次レビュー時**: 全体を確認・更新
4. **方針転換時**: 履歴を残して新しい方針を反映

### 更新担当

- **Orchestrator**: すべての更新を担当
- **Sub-Agents**: タスク完了をOrchestratorに報告
- **プロダクトオーナー**: Phase判断の承認
